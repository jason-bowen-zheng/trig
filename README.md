# TRIG
TRIG是一个最简三角方程、最简三角不等式求解器。

怎么说呢，有如下优点：

- 它是使用Python标准库写成的，无任何依赖
  > 精确的三角学对计算机来说确实是一个挑战
- 它与学校的教学完全相同
  > 是作者的，不是你的（为了满足老师对解集的要求，此程序多次改版）

<details>
<summary>方程、不等式的解法</summary>
<p>

方程嘛，挺简单的。大家都背过公式吧，套公式就可以了。

不等式嘛，利用单位圆或函数图像吧：`sin`横截单位圆；`cos`纵截单位圆；其它的看函数图像。
> 确实挺好实现的呢，代码我都看不懂qwq
